# Bisu

A security maintenance application.

## Gemfile

### rspec

Needed for test-driven development. This gem will add:

  - rspec-core
  - rspec-expectations
  - rspec-mocks

To the install list.

## Using

```
bundle install;
bundle exec ruby view/console/client.rb help;
```

## Developing

```
bundle install
bundle exec rspec
```

Any contribution is welcome.
